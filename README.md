Pokemon HeartGold/SoulSilver themed RGB LED matrix clock. This project uses a 64x32 (4mm pitch) RGB LED matrix board (from Adafruit), powered by a Raspberry Pi Zero W + Real Time Clock HAT.
Current features:
- Real-time digital clock display
- Animation chain including clouds, Pokemon and trainers
- Sun and moon positioning in the sky based on astronomical calculations using the astral library
